# 基于YOLOv5的疲劳驾驶检测系统 python928

## 项目概述

本项目是基于YOLOv5的疲劳驾驶检测系统，旨在通过深度学习技术对驾驶过程中的疲劳行为进行有效识别，如闭眼、打哈欠等，以提升驾驶安全性。

## 技术栈

- **主要框架：** Python
- **深度学习框架：** YOLOv5
- **用户界面：** UI界面（使用Qt框架）

## 功能模块

- **图片识别：** 选择图片进行疲劳行为识别
- **视频识别：** 对视频文件进行疲劳行为检测
- **实时检测：** 通过摄像头监控实时检测驾驶员疲劳状态

## 系统角色

- **用户：** 可进行登录、注册，并使用检测结果可视化功能

## 运行环境

- 环境配置指导视频包含于项目使用说明中

1. 环境配置

## 核心亮点

- **高效算法：** 使用YOLO v5算法，实现快速准确的目标检测
- **模型灵活性：** 支持一键切换更新模型
- **实时检测：** 实时监控并可视化检测结果
- **可扩展性：** 提供训练数据集和代码，可针对不同需求重新训练模型

## 原理简介

系统利用摄像机实时采集驾驶员面部特征，通过图像处理方法获取眼部、嘴部等特征信息。基于YOLOv5算法对这些特征进行分析，实时判断驾驶员的疲劳状态，并作出安全提示。主要检测的疲劳行为包括打哈欠和频繁眨眼等。
